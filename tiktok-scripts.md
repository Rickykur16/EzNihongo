# EzNihongo TikTok Content Playbook — 10 Script Pertama

**Target persona:** Mahasiswa Indonesia 18-25, sedang/mau push JLPT N5-N2, butuh tool murah + efektif.
**Channel:** TikTok @eznihongo (buat akun kalau belum ada — username: `eznihongo` atau `eznihongo.id`)
**Target KPI per video:** 5K+ views, 5%+ save rate, CTA click-through ≥ 1%.
**Durasi ideal:** 15-25 detik (algoritma TikTok suka completion rate tinggi).
**Rasio:** 9:16 vertikal.
**Aturan umum:**
- **3 detik pertama = nasib video.** Jangan self-intro ("Halo guys"). Langsung ke hook visual/verbal.
- **Text overlay wajib** (70%+ user nonton tanpa suara).
- **Captions TikTok** max 150 karakter (lebih dari itu kepotong).
- **CTA di pinned comment + link bio**, bukan cuma di caption (caption jarang dibaca).
- **Posting:** 3 video/hari selama 30 hari. Kalau 1 viral → 2 minggu konten cukup.

## VIDEO #4 — Edu hook: "Kanji yang mirip BANGET" (educational ⭐⭐⭐⭐⭐)

**Hook (0-3s):** Tampilin 4 kanji besar di layar: **日 目 白 自**. Text: **"LO BISA BEDAIN 4 INI?"**

**Body (3-22s):**
- Zoom satu-satu:
  - 日 (hi/nichi) = matahari / hari
  - 目 (me/moku) = mata
  - 白 (shiro/haku) = putih
  - 自 (ji/shi) = sendiri / auto
- Kasih trick untuk inget (contoh: "目 punya 2 garis horizontal = 2 pupil").

**CTA (22-25s):** *"Tips kayak gini ada di aplikasi gw. Link bio. N5 gratis."*

**Caption:** `4 kanji jebakan JLPT paling sering salah 👀 lo salah berapa? #kanji #jlpt #belajarkanji`

## 🎬 PRODUCTION KIT (peralatan minimal)

- HP aja (Android atau iPhone, sama aja).

- **Text overlay:** Noto Sans Bold untuk Indonesian text, Noto Sans JP untuk kanji.
- **Background music:** Pakai sound TikTok yg trending — JANGAN import musik luar (algoritma ga suka).
- **Lighting:** Lampu putih depan muka. Itu aja udah +50% quality.
- **Mic:** Built-in HP cukup asal ga rame.

## 📊 METRICS YANG DIPANTAU (via PostHog setelah dipasang)

Setelah user klik link bio dan sampai ke app:
- `page_view` — apa mereka nyampe?
- `session_active { logged_in: false }` — berapa % lanjut mainin N5?
- `level_click { level }` — N5 bounce atau main dalam?
- `login_modal_shown` — berapa yang kena paywall N4+?
- `install_banner_shown / dismissed / accepted` — berapa yang install PWA?

**Target Week 1:**
- 10 video × 5k avg view = 50k impressions
- CTR ke link bio: 1% = 500 kunjungan
- Install rate: 20% = 100 installs
- Active D1: 50% dari install = 50 user aktif

**Kalau kurang dari target:** iterate hook. Kalau lebih: scale angle yang menang.

---

## 🚫 YANG HARUS DIHINDARI

1. **Jangan** self-intro panjang ("Halo guys, nama gw..."). Scroll rate di 3 detik pertama = 70%.
2. **Jangan** terlalu sering nyebut brand di awal — bikin kesan iklan.
3. **Jangan** pakai suara AI / TTS robotic — viewer TikTok detect dan langsung skip.
4. **Jangan** posting dari akun baru langsung 10 video sehari — algoritma baca spam. Mulai 2-3 sehari minggu pertama, naik perlahan.
5. **Jangan** delete video yang flop — biarin aja, malah dump signal ke algoritma.

---

## 🔄 ITERATION LOOP

**Setelah 3-5 video pertama posting:**
- Cek view retention graph TikTok (analytics TikTok built-in).
- Identify video dengan retention >40% di detik ke-15 → itu **winner angle**.
- Remix winner 2-3 kali dengan variasi (topik beda, hook beda, sama format).

**Jangan** paksain posting template yang flop. Move on.
